jQuery require module loader

This js magic make client side module loader work like CommonJS or node js 'require' style, which load module synchronously.

The original idea was inspired by Torben Haase

Module script no longer need to wrap in a closure. Just 'exports' your module.

I use jQuery to simplified lots of the work without using Object.defineProperty, also leverage jQuery build-in cache. See, jQuery has all the power long long ago!!
